# PUMA Sources

No immutable external PUMA source snapshot is currently stored in this
package. That absence is intentional and is recorded as a gap rather than
turning legacy code comments into authoritative evidence.

| Source | Date/version | Use in TAVI | Status |
|---|---|---|---|
| `docs/INSTRUMENT_LAYOUT.md` | living project document | TAS conventions and historical PUMA geometry | comparison; not immutable evidence |
| `User_Guide.md` | living project document | operator-facing description of PUMA options | comparison; not immutable evidence |
| `components/README.md` | living project document | central custom component inventory | current dependency index |

Future external files belong in this directory with names of the form
`YYYY-MM-DD__source-id__vNN.ext`. Record publication/effective date, capture
date, edition, modeled fields, and conflicts here. Once captured, a source
snapshot is not edited; add a new version instead.
