# IN12 (ILL) — Research Package

IN12 is a cold triple-axis spectrometer. The retained upgrade paper describes
the long cold-neutron guide, velocity selector and guide changer, virtual
source, large double-focusing PG monochromator, sample position, reusable
secondary spectrometer, optional collimation, and detector-side capabilities.

This package is **not runnable in TAVI**. It contains research evidence only
and is absent from the registry. See [MODEL_STATUS.md](MODEL_STATUS.md) and put
expert corrections in [SCIENTIST_REVIEW.md](SCIENTIST_REVIEW.md).
