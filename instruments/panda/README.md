# PANDA (MLZ) — Research Package

PANDA is a cold triple-axis spectrometer. The available references describe a
supermirror guide and virtual-source concept feeding a focusing
monochromator, followed by the sample and secondary spectrometer. A historical
McStas instrument is retained as comparison material.

This package is **not runnable in TAVI**. It has no plugin or model and is not
registered. The source material has not yet been reconciled into a current
component inventory. See [MODEL_STATUS.md](MODEL_STATUS.md) and add any expert
comments to [SCIENTIST_REVIEW.md](SCIENTIST_REVIEW.md).
