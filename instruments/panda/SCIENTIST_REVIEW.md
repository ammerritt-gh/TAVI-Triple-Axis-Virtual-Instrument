# PANDA Scientist Review

Please comment anywhere in this file. A few corrections or a pointer to a
current drawing are more valuable than a completed form; TAVI maintainers will
do the structured extraction and coding.

The five highest-priority questions are:

1. Which current PANDA configuration should be the first TAVI model?
2. What are the source-to-mono, mono-to-sample, sample-to-analyser, and
   analyser-to-detector distances and axis senses?
3. Which monochromator/analyser crystals and focusing modes are currently
   available?
4. Which guide, virtual-source, collimator, slit, filter, and detector elements
   should appear in the beam path?
5. Is the retained `vpanda.instr` useful as historical truth, and which parts
   are now obsolete?
